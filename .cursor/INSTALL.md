# Installing Arxitect for Cursor

## Installation

To install the Arxitect skills in Cursor:

1. Clone the repository

```bash
git clone https://github.com/andonimichael/arxitect.git ~/.cursor/plugins/local/arxitect
```

2. Restart Cursor or run **Developer: Reload Window** to pick up the plugin.

## Updating

```bash
cd ~/.cursor/plugins/local/arxitect && git pull
```

## Uninstall

```bash
rm -rf ~/.cursor/plugins/local/arxitect
```
